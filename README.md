# Open-access Paper Analysis with GROBID

[![DOI](https://zenodo.org/badge/1179092077.svg)](https://doi.org/10.5281/zenodo.18966468)
[![CI](https://github.com/ghit2025/paper-analysis-grobid/actions/workflows/ci.yml/badge.svg)](https://github.com/ghit2025/paper-analysis-grobid/actions/workflows/ci.yml)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

Repository for the **Artificial Intelligence and Open Science** assignment at UPM.  
This project analyzes **10 open-access scientific articles** using **GROBID** and complementary PDF parsing.

---

## Objectives

1. Extract the **abstract** of each paper and generate a **keyword cloud**
2. Create a **visualization of the number of figures per article**
3. Produce a **list of links found in each paper**

---

## Installation

### Option A — Virtual environment

```bash
python3 -m venv env
source env/bin/activate
pip install -r requirements.txt
```

### Option B — Docker (recommended)

```bash
docker compose up --build
```

> Results will appear in the `results/` folder.

---

## Usage

### With virtual environment (requires GROBID running on port 8070)

```bash
python3 main.py --papers-dir papers --output-dir results \
  --grobid-url http://localhost:8070/api/processFulltextDocument
```

### With Docker Compose (starts GROBID automatically)

```bash
docker compose up
```

## Validation of results

### 1. Keyword cloud

The keyword cloud is generated only from abstracts extracted from GROBID TEI XML output.

- Manually checked that each TEI file contains an `<abstract>` section
- Compared extracted abstract text against the original PDFs for a sample of papers
- Confirmed the cloud excludes common stopwords

### 2. Figures per article

Figure counts are estimated from GROBID TEI `<figure>` elements, using labels such as `Figure 1` or `Fig. 2`.

- Manually inspected several TEI files and compared counts with the original PDFs
- Verified that non-figure labels (e.g. `Algorithm 1`, `Table 1`) are not counted
- Counts exported to `results/figures_per_article.csv` and visualized in `results/figures_chart.png`

### 3. Links per paper

Links are extracted from two sources:
- TEI references generated by GROBID
- PDF hyperlink annotations extracted directly from the original PDFs

Validation:
- Compared extracted links against original PDFs for several papers
- Confirmed duplicates are removed
- Verified output is grouped by paper in `results/links_per_paper.json`

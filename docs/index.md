# Open-access Paper Analysis with GROBID

This project analyzes 10 open-access scientific articles using **GROBID** and complementary PDF parsing as part of the **Artificial Intelligence and Open Science** course at UPM.

## What it does

- Extracts abstracts and generates a **keyword cloud**
- Counts figures per paper and creates a **bar chart**
- Extracts all **URLs/links** found in each paper
